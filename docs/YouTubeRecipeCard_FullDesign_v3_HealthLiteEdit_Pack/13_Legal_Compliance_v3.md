# 13 法務・規約・安全 v3（要点整理）

※法的助言ではありません。公開前に必ず一次情報を確認してください。

## 13.1 YouTube関連（最低限の運用方針）
- 動画/音声のダウンロード・保存をしない
- スクレイピングをしない（YouTube Data APIを利用）
- 埋め込みプレイヤーで視聴させる
- 元動画リンクとクレジットを常に表示
- API Data保持は必要最小限にし、更新日時（last_refreshed_at）を持って更新する

## 13.2 栄養推定の注意（健康/医療）
- 栄養値は推定であり誤差がある（油、量、個体差等）
- 医療助言ではない旨を表示
- 出典表記（食品成分データを利用する場合の出典）をアプリ内に表示
- 詳細栄養・疾病対応をMVPでやらない（高リスク）

## 13.3 共有ページの設計（リスク低減）
- 生成カードは“当社生成”として扱う
- ただし元動画を必ず埋め込み表示し、視聴の導線を提供する
