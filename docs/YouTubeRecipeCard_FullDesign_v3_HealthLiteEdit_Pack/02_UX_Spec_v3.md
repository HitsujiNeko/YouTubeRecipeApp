# 02 UX仕様 v3（栄養表示 + ちょい編集）

## 5.1 主要画面
1) Home: URL入力 + 最近のレシピ
2) ImportReview: 動画確認 + 生成開始
3) RecipeDetail: カード + 栄養 + 未確定リスト + 編集導線
4) CookMode: 調理モード（最重要）
5) NutritionFix: 未確定材料の“ちょい編集”画面
6) SharePage: 共有ページ（元動画＋カード＋栄養）

## 5.2 RecipeDetail（栄養カード）
- 上: タイトル、元動画リンク
- 中: 栄養（推定）カード
  - kcal / P / F / C / salt（1人前切替）
  - confidenceラベル（高/中/低）と coverage
  - 「未確定: 3件 → 直すと精度UP」ボタン
- 下: 材料・手順

## 5.3 NutritionFix（ちょい編集）
- 未確定材料を1件ずつカード表示
  - 元の材料行: 「鶏むね肉 1枚」
  - 候補: [鶏むね肉(皮なし)] [鶏もも肉] [鶏ささみ]（上位3）
  - 量入力:
    - 直接g（デフォルト）
    - もしくは単位選択（大さじ/小さじ/カップ）→内部でg換算
  - [保存して次へ]
- 最後に再計算し、RecipeDetailへ戻る

## 5.4 期待値管理（UIコピー）
- 栄養は推定である旨、誤差の可能性、医療助言ではない旨を明示
- confidenceが低い場合は「確定できた分のみで計算」などの説明

## 5.5 CookMode（継続の核）
- 手順：1ステップ表示（次/戻る）
- タイマー：手順に紐付け
- 材料：チェックリスト
- 動画：埋め込み（ミニプレイヤー）

## 5.6 Disclaimer（旧 17_UI_Copy_Disclaimer の統合）

### 栄養推定の注意（RecipeDetailに表示）
- 「栄養値は推定です。材料の分量や油の量などにより誤差が生じます。」
- 「医療・診断目的ではありません。体調や治療中の方は専門家にご相談ください。」
- 「未確定の材料を修正すると、推定精度が上がります。」

### 抽出の注意（Import完了時）
- 「AI生成のため誤りが含まれる可能性があります。元動画もあわせてご確認ください。」
- 「概要欄に材料がない動画は精度が下がります。」
