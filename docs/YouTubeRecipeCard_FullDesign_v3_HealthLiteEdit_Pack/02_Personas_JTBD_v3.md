# 02 ペルソナ / JTBD v3（健康ライト × ちょい編集B）

## 2.1 最重要ペルソナ P1（MVPターゲット）
**健康志向ライトの大学生/若手社会人**
- 属性:
  - 18〜30歳、一人暮らし比率高め
  - YouTubeでリュウジのバズレシピ/簡単レシピをよく見る
  - 健康目的：痩せたい/体型改善/筋トレ開始（いずれも「継続したい」）
- 既存の挫折:
  - 栄養管理アプリ（手入力）が続かない
  - YouTube調理は巻き戻しが多く疲れる
- Pain:
  - 作る前に栄養が分からない
  - 作ってみたら意外と高カロリー/脂質/塩分だった
  - 材料や手順を探すのが面倒
- Gain:
  - URLだけで「材料・手順・推定栄養」が出る
  - 調理中に“カード”で迷わない
  - 足りない部分だけ直せば（ちょい編集）栄養の信頼が上がる

## 2.2 準重要 P2（レシピ収集・整理したい層）
- Pain: 保存が散らかる、同じメニューに偏る
- Gain: タグ/検索/“高たんぱく”など健康フィルタで再発見できる

## 2.3 やらないペルソナ（MVPでは後回し）
- 厳密計測ガチ勢（誤差許容が低くサポートが重い）
- 医療目的（糖尿病/腎疾患等）中心（高リスク）

- 北極星: 「CookMode 完走率」×「栄養サマリー閲覧率」
- 主要KPI:
  - Import成功率（URL→カード生成成功）

  - “未確定材料の修正”完了率（ちょい編集が回っているか）
